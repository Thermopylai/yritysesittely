## :memo: Kuvaus

Tämä projekti esittelee yksinkertaisen kolmisivuisen yritysivuston, jonka muuttuva sisältö on tuotettu dynaamisesti **JavaScriptin** avulla. Fyysisiä **HTML-sivuja** on siis vain yksi kappale.

Sivustolla näytetään:
- yritysesittely
- yrityksen yhteystiedot
- yrityksen henkilökunta
- henkilökunnan tiedot tuotetaan **JSON-datan** perusteella

Ulkoasussa hyödynnetään **Bootstrapia**, ja rakenne on toteutettu semanttisilla **HTML-elementeillä**. Sivusto on muotoilultaan responsiivinen ja sivupaneelin pystyvalikko myös kollapsoituu. **JSON-data** tuodaan selaimelle **Express.js-palvelimelta**. Sivustolla on tumman ja vaalean väriteeman valinta valikossa sekä toimiva kello footerissa. Sivusto valitsee oletusteeman käyttöjärjestelmän asetusten perusteella ja valinta tallennetaan selaimen **localStorageen**.
